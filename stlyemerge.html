Product Overview

A Photoshop plugin that allows you to merge layer styles while preserving existing effects. Unlike the default paste behavior that replaces all effects, Style Merge adds new effects on top of existing ones, giving you complete control over which effects to combine.

Key Features:

• Preserve Existing Effects: Add new layer effects without losing the ones already applied
• Selective Effect Control: Choose exactly which effects to apply using checkboxes
• Drag-to-Select: Quickly select multiple effects by dragging across checkboxes
• Smart Effect Merging: Automatically combines multiple instances of the same effect type (e.g., multiple drop shadows)
• Effect Sorting: Intelligently sorts merged effects for consistent results
• Batch Apply: Apply copied styles to multiple selected layers at once
• Clear Function: Reset to initial state with one click

How to Use:

1. Select a layer with the style you want to copy
2. Click "Copy from Selected Layer"
3. Use checkboxes to select which effects to apply (or use Select All/Deselect All)
4. Select target layer(s) where you want to add the effects
5. Click "Paste to Selected Layer"

Important Notes:

• Existing enabled effects are preserved and merged with new ones
• Disabled effects in the target layer are ignored during merge
• Single-instance effects (Bevel & Emboss, Inner Glow, Outer Glow, Satin, Pattern Overlay) will replace existing ones if selected
• Multiple-instance effects (Stroke, Drop Shadow, Inner Shadow, Color Overlay, Gradient Overlay) are intelligently combined
• Supports undo functionality
